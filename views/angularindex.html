~ ANGULAR

Maximize your browser window. No more visual magic for you!

Angular is a MVC client-side framework for web developers. It's what HTML would have been, had it been built for dynamic content.

NG-FEATURES

    MVC
  • Unlike the famous jQuery, Angular gives us structure. It's almost impossible to deliver structured apps solely in jQuery. So separating code in modules becomes a necessity. Angular gives us MVC principles that we need for big projects.
    Dependency injection & Testing
  • These are great features. Rather than having to look for our dependencies, Angular does that work for you. Simply inject, for example a service into a controller, and Angular knows exactly what it is and what can be done with it. Dependency injection facilitates testing, which comes natively with Angular within a tool called Karma. TDD development becomes real.
    Directives
  • As I said before, Angular extends HTML. You can create new elements, new attributes, new components, taking advantage of directives. Create as many directives as you want. Turn your HTML upside down. Re-invent semantics.
