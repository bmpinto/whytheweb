~ CSS

CSS lets us do now, more than ever. It's almost a programming language thanks to these wonderful new frameworks.

Resize your browser window until you see something magical happen!

With CSS you can accomplish pretty much everything you can using Photoshop:

  • Drop shadows
  • Gradients
  • Strokes
  • Rounded corners
  • Shapes

But it has a lot more features that I love, as you can see by the animations so far.

All done with CSS.

It's not always rainbows and butterflies. It all comes with an expense. Browser support for recent features.

Frameworks to the rescue!

SASS/LESS give you the oportunity not to worry about prefixes. It also allows you to make sprites in a second and minify your CSS automatically.
